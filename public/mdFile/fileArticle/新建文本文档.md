冒泡

遍历n-1遍，比较相邻元素，将最大的冒泡到最后面

```javascript
export default (arr)=>{
    for(let i=arr.length-1,tem;i>0;i--){
        for(let j=0;j<i;j++){
            tem=arr[j]
            if(tem>arr[j+1]){
                arr[j]=arr[j+1]
                arr[j+1]=tem	
            }
        }
    }
    return arr 
}
```

选择

遍历n-1遍，每一遍遍历完成后将最小的置于最前面

```javascript
export default (arr)=>{
    for(let i=0,len=arr.length,min;i<len;i++){
        min=arr[i]
        for(let j=i+1;j<len;j++){
            if(arr[j]<min){
                let c=min;
                min=arr[j]
                arr[j]=c
            }
        }
        arr[i]=min
    }
    return arr
}
```
